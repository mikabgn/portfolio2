Projet SANCTIONS
Description

Le projet sanctions est un projet durant lequel ils nous avaient été demandé de créer un site de sanctions pour le lycée Gaudper. Dans ce site, il fallait avoir une page d'accueil, la possibilité de se connecter pour accéder aux différentes sections. Technologies utilisées

Bootstrap
HTML/CSS/SQL/Java Model MVC

  • Pouvoir importer des élèves depuis un csv
